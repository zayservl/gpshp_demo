## Точки синергии

1. **База НМД + Справочная система + Вопросно-ответный модуль** → «Корпоративный ИИ-поиск»
2. **Проверка комплектности + Анализ дублирующих заявок + Прогнозирование** → цепочка «заявка → проверка → планирование»
3. **Мемо совещаний (AI-юрист) + Автопротокол (AI-ассистент)** → «Автоматизация протоколов»
4. **Управление предложениями + Управление планами** → «Управление деятельностью» в ОИСМ
